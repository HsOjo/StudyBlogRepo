初等函数

定义

  • 由基本初等函数与常数经过有限次复合所构成。
  • 能用一个解析式表示。
    • 例如：$y = x^2 + cosx, y = e^{x^2}$都是初等函数。
    • 但函数$y = \begin{cases}e^x, x < 0 \ x^2 + 1, x >= 0\end{cases}$，是分段函数，不是初等函数。
