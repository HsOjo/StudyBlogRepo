什么是函数

定义

有数集$D ⊆ R$，如果存在某种对应法则$f$，可以使$D$中任意一个数$x$，都有唯一确定的数$y$ 与之对应（即结果唯一），则由$f$确定的$x$与$y$的对应关系可以记为：

$$y = f(x)$$

  • $f$: 函数
  • $x$: 自变量（即函数参数值）
  • $y$: 因变量（即函数返回值，别名：函数值）
  • $D$: 定义域（x的取值范围，$x ∈ D$）

如果$x_0 ∈ D$，则函数$f(x)$在$x_0$处有定义，在$x_0$处的函数（返回）值可以表示为：

  • $y|_{x = x_0}$
  • $f(x_0)$

没有声明定义域的函数，定义域默认为R

当$x$遍历完$D$，其中所有的$y$组成的数集便是值域，表示为：

$R_f = {y|y = f(x), x ∈ D}$

函数表达方式

  • 解析法/公式法

    • 如：$y = x^2$

  • 图形法

    • 如：使用直角坐标系表示的函数图像。

  • 表格法

定义域的表达

  • 参考集合的表达方式。

判断函数是否相同

  • 必要条件：定义域与对应法则（求$y$的解析式）完全相同。

定义域可以有多种等价的表达方式。

反函数

  • 设$y = f(x)$的定义域为$D_f$，值域为$R_f$

    • 如果$R_f$内的任意一个数$y$，根据$x = f(y)$都有唯一确定的数$x ∈ D$ 与之对应
      • 那么以$R_f$作为定义域的函数$x = f(y)$，便是$y = f(x)$的反函数。

  • 只有一一对应的函数才有反函数。

    • 如：$y = x^2$没有反函数
    • 如：$y = x^2, x ≥ 0$时的反函数为$y = \sqrt x$

如何求反函数

求反函数，只要得出以$y$表示的$x$，然后将$xy$互换即可。

  • 举例：$y = x^2 + 1, x ≥ 0$，那么$x = \sqrt {y - 1}$，即$y = \sqrt {x - 1}$。

特殊函数

  • 分段函数
    • 在定义域内不同的区间上对应法则用不同的解析式表示的函数。

$$
f(x)= \begin{cases}
0 & -1 < x ≤ 0\
x^2 & 0 < x ≤ 1\
3-x & 1 < x ≤ 2\
\end{cases}
$$
