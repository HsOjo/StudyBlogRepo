什么是集合

定义

  • 由一些元素组成的总体，叫做集合，简称集。
  • 集合内元素互不相同，即元素唯一。
  • 只要两个集合的元素是一样的，那么两个集合相等。

元素

通常使用大写字母A, B, C, …表示集合，而小写字母a, b, c, …表示元素。

如果a是集合A的元素，那么a属于A，即a ∈ A，否则a不属于A，即a ∉ A。

常用数集

  • $N$，非负整数集（自然数集）
  • $N^*$或$N_+$，正整数集
  • $Z$，整数集
  • $Q$，有理数集
  • $R$，实数集

点此查看更多。

表达方式

  • 列举法

    • 列出集合所有元素，并使用花括号括住。
      • A = {1, 2, 3}

  • 描述法

    • 根据元素特征进行描述，通常先定义一般符号及取值范围（如x ∈ R），再描述元素所具有的特征（即”|”后的内容）。
      • A = {x ∈ Z | x = 2k + 1, k ∈ Z}
      • A = {x ∈ R | x = $x^2 - 2 = 0$}

基本关系

  • 相等

    • 所有元素相同，即A = B。

  • 子集

    • 如果B包含的所有元素都在A中出现，那么A则是B的子集：A包含于B，即A ⊆ B，又或是B包含A，即B ⊇ A。
    • 子集可以与自身相等。
    • 空集是任何集合的子集。

  • 真子集

    • 与子集基本一致，A真包含于B，即A ⊊ B或A ⊂ B，又或是B真包含A，即A ⊇ B或A ⊃ B。
    • 真子集不可以与自身相等。

  • 空集

    • 不含任何元素，即∅。

基本运算

  • 并集

    • A与B包含的所有元素组成的新集合，即A ∪ B（A并B）。
    • A ∪ B = { x | x ∈ A，或 x ∈ B }

  • 交集

    • 所有同时在A与B的元素组成的新集合，即A ∩ B（A交B）。
    • A ∩ B = { x | x ∈ A，且 x ∈ B }

  • 补集

    • 全集：能包含问题中所有元素的集合，通常记作U。
    • A ⊃ U，那么属于U而又不属于A的元素组成的集合，便是集合A相对于全集U的补集，即$∁_UA$。
    • $∁_UA$ = { x | x ∈ U，且 x ∉ A }
