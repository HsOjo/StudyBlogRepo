基本初等函数之一

大纲

  • 幂函数
  • 指数函数
  • 对数函数

幂函数

名称：Power Function，简称pow

  • $y = x^µ$（$µ$为实数）
    • 定义域（注意是指$x$）：随$µ$而异。
      • 但无论$µ$为何值，$y = x^µ$在$(0, +∞)$内总有定义。
      • 其函数图像都通过点$(1, 1)$。
      • 当$µ = 0$时，$y = 1$是常函数。
  • 图像：

  • 性质：
    • 当$µ > 0$，第一象限（右上）内，函数单调递增，如$y = x, y = x^2, y = x^\frac 1 2$（即图1所示）。
    • 当$µ < 0$，第一象限内，函数单调递减，如$y = x^{-1}, y = x^{-2}$（即图2所示）。

指数函数

名称：Exponential Function，简称exp

  • $y = a^x (a > 0, 且 a ≠ 1)$
    • 定义域：$(-∞, +∞)$（即$R$）
      • 值域：$(0, +∞)$
      • 其函数图像都通过点$(0, 1)$
  • 图像：

  • 性质：
    • 当$a > 1$时，函数单调递增（如图1所示）
    • 当$0 < a < 1$时，函数单调递减（如图2所示）
    • 当$a = e$时（$e = 2.71828…$，为无理数），函数为$y = e^x$，在极限中会用到。
  • 常见运算法则
    • $\frac {a^m} {a^n} = a^{m-n}$
    • $a^m * a^n = a^{m+n}$
    • $(a^m)^n = a^{mn}$
    • $\sqrt[n] {a^m} = a^\frac m n$

对数函数

名称：Logarithmic Function, 简称log

  • $y = log_ax (a > 0, a ≠ 1)$
    • 定义域：$(0, +∞)$
      • 值域：$(-∞, +∞)$
      • 其函数图像都经过点$(1, 0)$
  • 图像：

  • 性质：
    • 当$a > 1$时，函数单调增加（如图1所示）
    • 当$0 < a < 1$时，函数单调减少（如图2所示）
    • 当$a = e$时，即以$e$为底的对数函数称为自然对数，记为$y = lnx$。
    • 当$a = 10$时，即以10为底的对数函数称为常用对数，记为$y = lgx$。
  • 常用运算法则（$a > 0 且 a ≠ 1$，$b > 0 且 b ≠ 1$，$c > 0 且 c ≠ 1$，$M > 0, N > 0$）
    • $log_aa = 1$
    • $log_ab = \frac 1 {log_ba}$
    • $log_aM + log_aN = log_aMN$
    • $log_aM - log_aN = log_a\frac M N$
    • $log_aM^x = xlog_aM$
    • $b = a^{log_ab}$
    • $log_ab = \frac {log_cb} {log_ca}$
    • $a^x = e^{xlna}$
    • $a^x = N ⇔ x = log_aN$

补充：

  • $log_{10}x = lgx$
  • $log_ex = lnx$
  • $log_ax = \frac {lnx} {lna}$
