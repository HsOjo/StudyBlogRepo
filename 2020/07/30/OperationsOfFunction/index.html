函数的运算

大纲

  • 四则运算
  • 复合运算

四则运算

  • 设有两个函数$f(x), x ∈ D_1$与$g(x), x ∈ D_2$，并设$D=D_1∩D_2, D ≠ Ø$，那么$f(x), g(x)$在$D$上定义的求和（加法）、求差（减法）、求积（乘法）运算如下：

    • 求和运算：$F(x) = f(x) + g(x), x ∈ D$
    • 求差运算：$G(x) = f(x) - g(x), x ∈ D$
    • 求积运算：$H(x) = f(x)g(x), x ∈ D$

  • 由于求商（除法）运算需要确保除数不为0，那么则令$D^* = D_1 ∩ {x | g(x) ≠ 0, x ∈ D_2} ≠ Ø$，而$f(x), g(x)$在$D^*$上定义的求商运算如下：

    • 求商运算：$L(x) = \frac {f(x)} {g(x)}, x ∈ D^*$

如果$D = D_1 ∩ D_2 = Ø$，那么则不能进行四则运算。

复合运算

  • 设函数$y = f(u)$的定义域为$D_f$，函数$u = g(x)$的定义域为$D_g$，且值域$R_g ⊆ D_f$，那么：
    • 可以得到函数：$y = f[g(x)], x ∈ D_g$
      • 该函数为$u = g(x)$与$y = f(x)$构成的复合函数，定义域为$D_g$，而$u$为中间变量。

即函数套娃，被套娃的函数，值域必须是套娃函数的定义域的子集。
